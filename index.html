About

Strong years of experience as a .NET Web Developer. Responsible for design and implementation of features, resolving defects, creating reports and technical documentation for web applications. Technical management of junior developers. I have been a key resource in several long-term, challenging and time critical projects. My high quality deliverables have secured follow up work from several clients.

I have substantial experience in web development using the .NET Framework, C#, VB.NET, WCF, LINQ, Silverlight, SQL Server 2005/2008/2012, and SQL Server Reporting Services. Other technical skills include C/C++, ASP.NET, JavaScript, Windows Phone, jQuery, Java, CSS, Oracle DB, PL/SQL, and Crystal Reports.
